ICG Filter

Данная программа предназначена для применения различных фильтров к изображениям.

Основные возможности:

  • Гамма-коррекция
  • Чёрно-белый фильтр
  • Инверсия цвета
  • Фильтры выделения границ
  • Фильтр удаления артефактов
  • Фильтр установки источника света
  • Размытие по Гауссу
  • Упорядоченный дизеринг и дизеринг Флойда-Стенберга
  • Фильтр резкости
  • Акварельный фильтр
  • Фильтр с произвольной матрицей

Задание параметров фильтров производится в интуитивно понятном диалоговом окне. Все фильтры:

  • Дублированы в меню для удобства доступа
  • Снабжены всплывающими подсказками

Дополнительные функции:

  • Возможность переключения между исходным и отфильтрованным изображением
  • Автоматическая подгонка изображения под размер экрана
